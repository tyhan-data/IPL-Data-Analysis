# 🏏 IPL Data Analysis | Exploratory Data Analysis Project


---

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on Indian Premier League (IPL) match data to uncover insights about team performance, toss decisions, venue trends, and player achievements.

The goal of this project is to practice real-world data analysis using Python and develop data storytelling skills through visualization and insights.

---

## 🎯 Objectives
- Understand IPL match patterns
- Analyze team and player performance
- Study toss decision impact
- Explore venue-based trends
- Generate meaningful insights using visualizations

---

## 📊 Dataset
The dataset contains historical IPL match information including:

- Teams and match winners
- Toss results and decisions
- Match venues
- Player of the Match awards
- Match outcomes and margins

---

## 🛠 Technologies Used
- **Python**
- **Pandas** — Data manipulation
- **NumPy** — Numerical operations
- **Matplotlib** — Data visualization
- **Seaborn** — Statistical visualization
- **Jupyter Notebook**

---

## 🔍 Analysis Workflow
The project follows a structured data analysis pipeline:

1. Data Loading & Exploration
2. Data Cleaning & Preprocessing
3. Team Performance Analysis
4. Toss Impact Analysis
5. Venue Analysis
6. Player Performance Analysis
7. Score Distribution Analysis
8. Insight Generation

---

## 📈 Key Insights
- Gujarat Titans have recorded the highest number of match wins in the dataset.
- Mumbai Indians and Chennai Super Kings have comparatively fewer wins.
- Teams generally prefer fielding after winning the toss, indicating a chasing advantage.
- Wankhede Stadium, Mumbai has hosted the highest number of matches.
- Eden Gardens has hosted the lowest number of matches among the venues analyzed.
- Quinton de Kock recorded the highest individual score.
- A small group of players dominate the Player of the Match awards, showing consistent individual performance.
